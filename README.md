# ⚔️ Caravan Hero

### Sobre o Projeto

**Caravan Hero** é um jogo **RPG tático por turnos com elementos de Roguelite**.

O conceito gira em torno do gerenciamento estratégico de um esquadrão de heróis. O jogador deve recrutar mercenários de diferentes classes (Guerreiro, Mago, Tanque, etc.), posicioná-los no campo de batalha e enfrentar ondas infinitas de inimigos gerados proceduralmente. O jogo conta com sistemas de:

* **Combate por Turnos:** Baseado em Iniciativa e Velocidade.
* **Economia:** Ganhe ouro, contrate novas unidades na loja.
* **Progressão:** Sistema de Nível, Atributos (Força, Agilidade, etc.) e "Rebirth" (Tier).
* **Permadeath:** Se uma unidade morre, ela é perdida para sempre.

### Tecnologias Utilizadas

O projeto foi desenvolvido inteiramente com tecnologias nativas da Web (Vanilla), sem o uso de frameworks externos, focado em Orientação a Objetos e padrões de projeto (Factory, Singleton, Managers).
